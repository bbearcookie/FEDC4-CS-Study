# padding vs margin

<img src = 'https://blog.kakaocdn.net/dn/bbACit/btqNVSuhdVz/jYMQZnNpH8LZc2EY326po0/img.png'/>

## padding

Object 내의 내부 여백 -> 요소 내부에 빈 공간을 만듬<br/>
음수 값이 유효하지 않으며 `auto`도 사용 불가함

background-color 속성으로 설정하는 배경색의 영향을 받음

## margin

object와 화면과의 여백(외부 여백) -> object 밖의 주변 공간을 추가함<br/>
block요소(ex.div)에만 적용되고 inline요소(ex. span)에는 적용되지 않음<br/>
음수 값이 유효하며 `auto`도 사용 가능

background-color 속성으로 설정하는 배경색의 영향을 받지 않음
